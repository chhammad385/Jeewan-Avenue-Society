🎯 Role-Based Permissions

💰 Finance Secretary

Can Access: ALL financial operations (add, edit, delete financial records)
Cannot Access: Announcements, Member Management, Document Uploads, Plot Management
→ Buttons will be disabled on these pages

📢 Information Secretary

Can Access: Create announcements, Upload society documents, Set prayer timings
Cannot Access: Financial operations, Member Management, Plot Management
→ Buttons will be disabled on these pages

📋 General Secretary

Can Access: Member management (add/edit/delete), User management, Plot management (edit/save/delete), Renter docs (add/edit ONLY General Secretary)
Cannot Access: Financial operations, Create announcements, Upload documents
→ Buttons will be disabled on these pages

👑 President

Can Access: Member management (add/edit/delete), User management, Plot management (edit/save/delete)
Cannot Access: Financial operations, Create announcements, Upload documents, Renter docs editing
→ Buttons will be disabled on these pages
Note: President can see renter docs but cannot add/edit (General Secretary only)

🤝 Vice President & Others

View-Only Access: Can access President Console pages but cannot modify anything
→ All action buttons will be disabled

⏰ Prayer Timing Exception

Special Rule: ALL roles can set prayer timings
→ Prayer timing buttons remain enabled for everyone

🔧 How It Works

  • Automatic Detection: System detects user role when page loads
  • Smart Button Disabling: Identifies and disables unauthorized buttons
  • Professional Styling: Grayed-out buttons with "not-allowed" cursor
  • Helpful Tooltips: Explains what role is required for each action
  • Multiple Protection Layers: Frontend + Backend security
  • Dynamic Content Support: Works with buttons added later via JavaScript
